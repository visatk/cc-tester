# 💳 cc-tester

> Zero-dependency, highly performant credit card testing utility built for Cloudflare Workers, Pages, and the Edge.

[![npm version](https://badge.fury.io/js/edge-cc-tester.svg)](https://badge.fury.io/js/edge-cc-tester)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Stop searching for Stripe test cards. `cc-tester` provides instant access to mock credit cards for various testing scenarios (success, declines, fraud) along with ultra-fast Luhn validation and network detection. 

## Features
- ⚡ **Edge-Ready:** 100% compatible with Cloudflare Workers, Vercel Edge, and Deno.
- 🪶 **Zero Dependencies:** Tiny footprint, no bloat.
- 🛡️ **TypeScript Native:** Full type safety out of the box.

## Installation

```bash
npm install cc-tester
# or
pnpm add cc-tester
